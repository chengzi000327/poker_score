# 烧鸡记分微信小程序（MVP）

用于 4 人烧鸡打牌的每局记分与累计统计。

## 已实现功能

- 支持设置 4 位玩家名字
- 支持设置筹码单价（元）
- 每局录入每人：
  - 奖数量（1 奖 => 其余 3 人各给 1）
  - 烧鸡数量（1 烧鸡 => 自己给其余 3 人各 1）
- 每局可重新组队（2v2）
- 每局可设置输方（A组/B组）与输档（1 或 2）
- 一键结算本局净筹码、净金额
- 一键生成“谁给谁多少钱”最少转账清单
- 保存历史并显示累计金额

## 规则口径（当前版本）

- 奖净筹码：`3 * 本人奖 - 其他三人奖之和`
- 烧鸡净筹码：`其他三人烧鸡之和 - 3 * 本人烧鸡`
- 组队净筹码：
  - 输方每人：`-2 * 输档`
  - 赢方每人：`+2 * 输档`
- 本局净筹码：`奖净筹码 + 烧鸡净筹码 + 组队净筹码`
- 本局净金额：`本局净筹码 * 筹码单价`

## 本地运行

1. 打开微信开发者工具
2. 选择“导入项目”
3. 目录选择本项目根目录
4. AppID 可先使用测试号（`touristappid`）
5. 编译后即可使用
